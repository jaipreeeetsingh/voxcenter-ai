# VoxCenter AI

The AI call center platform that **actually works**.

Built by analyzing 2,000+ complaints about Retell, Vapi, Bland, Synthflow and Air AI.

## Why VoxCenter?

| Problem | Others | VoxCenter |
|---------|--------|-----------|
| Latency | 600ms-2s | Under 300ms |
| Hallucinations | No guardrails | Real-time fact-checking |
| Testing | Test in production | Full test suite |
| Pricing | Hidden fees | 100% transparent |
| Lock-in | Single provider | Swap with 1 click |
| Minutes | Expire monthly | Roll over forever |

## Quick Start

1. Clone: `git clone https://github.com/jaipreeeetsingh/voxcenter-ai.git`
2. 2. Install: `cd voxcenter-ai && npm install`
   3. 3. Configure: `cp .env.example .env` and add your API keys
      4. 4. Database: `npx prisma db push`
         5. 5. Run: `npm run dev`
            6. 6. Open: http://localhost:3000
